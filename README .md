Auto Rhythm Video Editor is a Python-based project that creates music-synced videos
by extracting frames from a video and merging them with audio.
A Streamlit web app allows users to upload video and audio files easily.

## Features
- Upload video and optional audio
- Extract frames using OpenCV
- Merge audio and video using FFmpeg
- Simple Streamlit web interface
- Fast and automatic processing

## Tech Stack
- Python
- OpenCV
- FFmpeg
- MoviePy
- Streamlit

## How to Run

1. Clone the repository

2. Create virtual environment

3. Install dependencies

4. Run the app

## Output
- Extracted frames
- Final video with audio
